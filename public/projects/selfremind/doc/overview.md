# SelfRemind
SelfRemind is a selfhosted To-do List Manager built with [Next.js](https://nextjs.org/), [Chakra-UI](https://chakra-ui.com/) and [MySQL](https://www.mysql.com/).

## Features
- User Authentication
- Create, Update and View To-do Lists
